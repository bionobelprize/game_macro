üöÄ Quick Start

Installation

# Clone the repository
git clone https://github.com/bionobelprize/game_macro.git
cd game_macro

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py

Recording a Macro

  1. Launch the application with python main.py
  2. Click the "Start Recording" button
  3. Perform your mouse movements and keyboard actions
  4. Click "Stop Recording" when finished
  5. Enter a name for your macro (e.g., "Combat Combo")
  6. Optionally set a hotkey (e.g., "ctrl+shift+1")
  7. Click "Save Macro" to save

Playing a Macro

Method 1: Using the GUI

  1. Select your macro from the list
  2. Adjust playback speed if needed
  3. Click "Play Macro"

Method 2: Using Hotkeys

  1. Assign a hotkey to your macro (e.g., "ctrl+shift+1")
  2. Press the hotkey anywhere to trigger the macro

Architecture

  • macro_recorder.py - Core recording and playback engine
  • macro_gui.py - Tkinter-based graphical interface
  • main.py - Application entry point

Dependencies

  • pynput - Cross-platform mouse and keyboard control
  • keyboard - Global hotkey support
  • tkinter - Built-in GUI framework (included with Python)

File Format

Macros are stored as JSON files with event data:

{
"timestamp": 0.5,
"type": "mouse_click",
"data": {
"x": 500,
"y": 300,
"button": "left",
"pressed": true
}
}
